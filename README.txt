Валидный формат входного файла: архив, содержащий article.txt (
                                первая строка - заголовок, вторая - тема статьи (Путешествия, Спорт, Экономика, Политика),
                                третья строка - текст статьи)


Для запуска проекта на Windows:

1. Загрузите файлы из репозитория (Используйте Git, команда git clone https://github.com/NatD88/News)

2. Загрузите и установите Maven
  2.1 Загрузите файл для установки Maven  с официального сайта https://maven.apache.org/download.cgi
  2.2 Добавьте переменные среды для Maven. Для этого в "Этот компьютер" щелкните правой кнопкой мыши,
  выбирете Свойства - Дополнительные параметры системы - Переменные среды.
  В разделе "Переменные среды для пользователя //user" 
     - нажмите "Создать", добавьте переменную H2_HOME и укажите путь к директории, где установлен Maven
     - выбирете переменную Path, нажмите "Изменить" -> "Создать", добавьте строку "%M2_HOME%\bin",
     также проверьте наличие переменной JAVA_HOME (она должна указывать пуьб к директории с jdk)
  В разделе "Системные переменные" выполнить те же действия

3. Откройте командную строку, перейдите в папку со скачанным проектом и выполните команду "mvn clean install"

4. Файл с расширением .war из папки target переименуйте в news.war

5. Загрузите и установите Tomcat  с официального сайта https://tomcat.apache.org/download-90.cgi
     Добавьте переменные среды для  Tomcat
        CATALINA_HOME = путь к директории с Tomcat, 
        CLASSPATH = путь к директории библиотек (..\Tomcat 9.0\lib)
        В переменную Path добавить:
             %CLASSPATH%\servlet-api.jar
             %CATALINA_HOME%\bin
             %CLASSPATH%\servlet-api.jar
   Проверьте адреса портов в файле server.xml (..\Tomcat 9.0\conf\). Connector port="8080", <Server shutdown="SHUTDOWN" port="8005">
     
6. Скопируйте файл news.war в Tomcat /webapps

7. Запустите Tomcat

8. Перейдите по ссылке http://localhost:8080/news/articles
